Open MOS Backend
============================================

This is the documentation for the Open MOS Backend.

MOS is a universal *Mathematical Optimization Service* that facilitates the deployment, integration, management and utilizati\
on of mathematical optimization models. Open MOS Backend, once
launched, provides a server
that allows an optimization model be accessible via a REST API.
